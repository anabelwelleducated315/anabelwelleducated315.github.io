⚙️ modusignal - Debug industrial devices in your browser

Download modusignal

Modusignal provides a centralized platform for testing and configuring industrial hardware. You can manage serial ports, WebUSB, WebSocket, MQTT, Modbus RTU, and HART protocols from one interface.

🚀 Getting Started

This software simplifies how you interact with machines. You do not need complex backend servers or heavy installations. Since this tool runs inside your web browser, it works across different device types without requiring deep technical knowledge.

📥 Installation Guide

Follow these steps to set up the software on your Windows computer:

  1. Visit the official download page.
  2. Locate the latest release version designated for Windows.
  3. Initiate the download for the installer package.
  4. Once the download completes, open the file.
  5. Follow the on-screen prompts to finish the setup process.

🔧 Key Features

  • Serial Communication: Connect directly to physical serial ports for data monitoring and command delivery.
  • Modbus RTU Support: Read from and write to Modbus devices with a simple visual interface.
  • WebUSB Integration: Hook up USB industrial devices directly to your web interface.
  • Protocol Flexibility: Use custom protocols if your specific equipment requires unique handling.
  • MQTT Connectivity: Push and pull data from networked IoT devices seamlessly.

💻 System Requirements

To ensure steady performance, your computer should meet these standards:

  • Windows 10 or Windows 11.
  • A modern web browser such as Chrome or Edge.
  • At least 4GB of RAM for smooth data processing.
  • A stable internet connection for web-based features.

🛠 Troubleshooting

If you cannot see your device, ensure the hardware is plugged into your USB port correctly. Refresh your browser page to reload the hardware detection service. Check that other programs are not currently locking the serial port you wish to access, as only one application can control a serial device at a time.
